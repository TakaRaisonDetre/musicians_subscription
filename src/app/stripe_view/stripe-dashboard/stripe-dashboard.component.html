Stripe Dashboard

This page demonstrates how to retrieve data from Stripe.

Charge History

Retrieves the customer charge history from Stripe.

Component Usage: <user-charges></user-charges>

Payment Sources

Retrieves the customer payment sources from Stripe.

Component Usage: <user-sources [canSelect]="true"></user-sources>

When canSelect == true, the component will display a button for each source and emit selectedSource event - this is useful for sharing data between components.

Subscriptions

Retrieves the customer subscriptions from stripe.

Component Usage: <user-subscriptions></user-subscriptions>
